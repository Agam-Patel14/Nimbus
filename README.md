# Nimbus

Nimbus is an AI-powered design tool that enables users to effortlessly create stunning logos and posters. By leveraging advanced AI models, Nimbus transforms simple text prompts into visually appealing designs, making it easy for anyone to produce professional-quality graphics without prior design experience.

## Features
- **AI-Generated Logos**: Create unique and professional logos based on user-defined text prompts.
- **AI-Generated Posters**: Design eye-catching posters with ease using AI-driven image generation
- **User Authentication**: Secure user registration and login system.
- **Activity History**: Track and manage previously generated designs.
- **Responsive Design**: User-friendly interface that works seamlessly across devices.
- **Scalable Backend**: Built with Node.js and Express for robust performance.
- **Modern Frontend**: Developed with React for an intuitive user experience.
- **Database Integration**: Utilizes MongoDB for efficient data storage and retrieval.
- **Cloud-Based AI Services**: Integrates with Hugging Face for powerful AI capabilities.
- **Email Generation**: Automated email drafting for professional communicaton.
